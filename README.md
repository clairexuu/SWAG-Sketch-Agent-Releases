# SWAG-Sketch-Agent-Releases

An AI-powered design assistant that converts natural language ideas into rough, draw-over-ready concept sketches. Built for Swag's licensed design pipeline, it uses retrieval-augmented generation (RAG) with designer-curated reference images to produce sketches that match exact stylistic guidelines — accelerating early-stage concept throughput while preserving designer control.

## Getting started

### Create an account

Use **Sign up** with your **invitation code**, **email**, and **password** (at least eight characters). After registering, sign in from **Login** when you open the app.

### Configure API keys

Open **Dashboard** in the top navigation (your account and settings live here). Add your **Google Gemini** and **OpenAI** API keys — both are required for the default sketch workflow. You can save and test keys on this page. Other provider keys may appear here depending on product updates.

### Studio

Open **Studio** (the home view). **Select a style**, describe what you want in the prompt, and generate sketches. Refine or iterate using the controls shown in the workspace.

### Styles

Open **Styles**. Pick a style from the list to view or edit it.

- **Visual rules:** Choose **Edit**, then set **Line Weight**, **Looseness**, and **Complexity**, plus **Name** and **Description** as needed, and save.
- **Reference images:** Use **Add Images** to upload. To remove images, select them in the gallery, then **Delete Selected**.

### Archive

Open **Archive** to browse past sketch generations. **Only generations from the past 14 days are kept** — older runs are not stored.
